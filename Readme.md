Game & Sofware Developer
-----------------------

I am a Full Stack Developer with a passion for both game development and 3D design. With experience in Godot, Blender, React, Node.js, and Firebase, I have built a range of applications, including advanced game mechanisms for FPS and monster cars, as well as creating immersive Godot environments like forests and various scenes.

In addition to programming, I excel in designing smooth user experiences and efficient backends. I am driven by continuous learning and teamwork, having led project teams and contributed to open-source projects in both the Godot and Blender communities.

* 🌍  I'm based in India
* 🖥️  See my other github account at [Github](https://github.com/Lakshman-kumar-121/)
* 🚀  I'm currently working on [Mechanism of Godot](http://github.com/Lakshman-YT)
* 🧠  I'm learning GLSL Shaders
* 🤝  I'm open to collaborating on Game dev Projects
* ⚡  Contributed to the Godot Engine by implementing in-engine logic and creating tutorials for 3D space. Enhanced functionality for developers and collaborated with the community to improve documentation and support.
